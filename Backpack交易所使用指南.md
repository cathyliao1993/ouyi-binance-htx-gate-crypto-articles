# Backpack交易所使用教程

本文旨在指导用户如何在OKX交易所在香港受限后，安全提取其中的数字资产。

尽管OKX交易所限制了香港用户的使用权限，但提币功能依然开放。首先，访问OKX官网（可通过浏览器搜索OKX.com）。若页面提示IP受限，请更换IP地址至中国台湾、日本或韩国等地区，避免使用美国和加拿大的IP。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b200b4be6e_1760959685-110.png)

# **往钱包提币**

登录OKX后，进入【资产管理】页面，选择【提币】选项。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/96296d8f6c_1760959699-111.png)

选择目标币种（如USDT），并在网络选项中指定Arbitrum one网络。随后输入充币地址。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/55cc412a53_1760959708-112.png)

例如，若需将资产转至币安钱包，在币安钱包界面选择【接收】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/88d8978a26_1760959720-113.png)

币种选择“USDT”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/77c9a41d3a_1760959730-114.png)

网络同样选择Arbitrum one。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/07e68415c9_1760959739-115.png)

注意：网络必须一致，否则资产可能丢失。复制币安钱包地址，粘贴至OKX提币界面。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/dbe322b322_1760959747-116.png)

输入提币数量（如10 USDT），扣除0.098网络手续费后，到账金额为9.902 USDT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e47e94ce45_1760959756-117.png)

也可选择到账10 USDT，系统自动扣除10.098 USDT，点击【下一步】完成操作。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/92d9fba69b_1760959766-118.png)

此方法可将OKX资产分批转移至币安钱包。除USDT外，其他币种也支持提币。务必确保充提网络匹配，并预留Gas费。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0769adb70d_1760959774-119.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/66aa2a32d4_1760959784-120.png)

# **往交易所提币**

许多用户不习惯钱包操作，且个人托管风险较高（如黑客攻击）。对于现货交易者，币安交易所是可靠选择。

访问币安官网（binance.com）。若遇IP限制，通过VPN切换至中国台湾、日本或韩国IP，使用香港ID注册认证。认证后账户支持除合约外的所有功能。点击右上角【充值】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2fd5d43e4d_1760959793-121.png)

选择【数字货币充值】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/23b9dd12f9_1760959806-122.png)

选择币种（如USDT）和网络（Arbitrum one），复制显示的充币地址。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/dd60317652_1760959817-123.png)

返回OKX粘贴地址，输入提币数量，点击【下一步】逐步转移资产。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7e3e149895_1760959845-124.png)

币安仅支持香港用户现货交易。如需合约交易，推荐Backpack交易所，其核心优势是合规性（持有欧盟加密货币衍生品唯一牌照）。Backpack的透明度和合规性优于Binance、OKX、Bitget、Bybit等平台。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4eb578109d_1760959854-125.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ba946ec243_1760959862-126.png)

进行身份认证（合规要求），点击【个人】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4334e281bc_1760959870-127.png)

居住国家选择“hongkong”，按香港ID如实填写姓名和出生日期。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/68e9d86551_1760959878-128.png)

城市填“hongkong”。资金来源选“就业收入”，就业状态选“在职”（无需额外证明）。非美国公民或绿卡持有者选“否”，点击下一步。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/db452f231a_1760959886-129.png)

点击继续。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/493bb88fe3_1760959894-130.png)

ID持有地区搜索“hongkong”，支持香港ID认证。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/731845d8e6_1760959903-131.png)

上传香港ID证件或通过手机进行人脸识别。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a060497acf_1760959911-132.png)

认证成功后即可登录Backpack。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1e6ac8ce11_1760959919-133.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5cbf070cb7_1760959959-137.png)

币种选USDT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/81f0c6c84f_1760959969-138.png)

网络选Arbitrum one，粘贴地址，输入数量（如10 USDT），点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/75c0746864_1760959978-139.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d8064d71fb_1760959990-140.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9aeab3b102_1760960002-141.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9bf8aa7c85_1760960020-143.png)

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c4ba4403ca_1760960029-144.png)

# **进行合约交易**

合约交易以USD为基准，确保合规透明。选择品种（如索拉纳合约）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d940f192a9_1760960048-146.png)

设置杠杆（最高50倍），如选5倍，点击【确认】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b4c657c7dd_1760960056-147.png)

启用闪电按钮的自动出借功能，账户闲置资金和保证金自动生息。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a064ea069e_1760960068-148.png)

左侧为K线图区域。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a874c93428_1760960079-149.png)

图表形式查看资金费用。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0022d093b9_1760960090-150.png)

右侧为代币信息，确认后选择做多或做空。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6c180d8ad3_1760960101-151.png)

例如，做多索拉纳合约：在操作区设置杠杆，拖动滑杆选择数量（可切换为美金或代币单位），点击【买入做多】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/daa14a8fd6_1760960110-152.png)

持仓显示在下方，盈亏基于标记价格（整合Coinbase、币安、OKX等平台价格），避免异常波动爆仓。显示预估强平价、保证金和盈亏。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/abbbc69251_1760960118-153.png)

限价平仓：输入目标价（如230），点击【卖出做空】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/bc943fa021_1760960128-154.png)

市价平仓：选择“市价”，滑杆拉满，点击【卖出做空】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9c9c3d1a1c_1760960137-155.png)

订单关闭。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b1bbd73269_1760960146-156.png)

支持止盈止损设置。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/644a043df1_1760960957-4.png)

启用自动出借后，资金自动收益。USD收益最高。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/541b8d09d2_1760961607-5.png)

透明借贷模型：USD借贷池显示使用率（如86.6%时利息达40%），反映市场趋势。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c097530c56_1760961685-6.png)

APP下载：在【更多】选择【下载】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a31011c95f_1760961774-7.png)
